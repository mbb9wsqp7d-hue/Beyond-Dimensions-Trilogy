# 《维度之外三部曲》小说数据库

欢迎来到《维度之外》三部曲的官方设定数据库。本仓库旨在通过结构化的方式，管理这一宏大科幻史诗的世界观、人物、剧情及底层逻辑。

## 📖 三部曲架构

| 部次 | 书名 | 核心主角 | 核心主题 | 哲学命题 |
| :--- | :--- | :--- | :--- | :--- |
| **第一部** | **《众生游戏》** | **易小星** | **人性觉醒** | 虚构生命是否具有真实的人性？ |
| **第二部** | **《时墟》** | **陆辰** | **数据人权** | 意识上传后，数据生命是否有尊严？ |
| **第三部** | **《超维之域》** | **林凡** | **文明归宿** | 多维宇宙中，文明的终极去向何方？ |

## 📂 目录说明

- **`/Settings`**: 包含世界观、九大文明、Ω协议等底层逻辑设定。
- **`/Characters`**: 包含三代主角及关键配角（程汐、赵教授等）的详细档案。
- **`/Plot`**: 包含三部曲的详细章纲与剧情逻辑。
