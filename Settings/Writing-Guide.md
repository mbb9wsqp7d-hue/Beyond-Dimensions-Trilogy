# 写作系统与叙事逻辑深度解析提取

## 1. 系统语言风格总库
系统语言需保持“冷硬中立、无情绪、结构美感、逻辑秩序”的特征。

### 核心提示模板
- **初始化/状态**：【系统初始化完成——核心加载中】、【核心中枢运行稳定】。
- **采样/数据**：【检测到情绪波动】、【采样完整度：XX%】、【行为样本：XX入档】。
- **功法/进阶**：【功法识别：XX·凡级】、【检测到功法自演】、【宿主心识跃迁】。
- **法则/宇宙**：【逻辑碎屑 → 法则残片】、【法则融合率 XX%】、【检测到自演律激活】。

## 2. 三重叙事写作模式 (System-Host-Civilization)
在同一章节中同步运行三种语言逻辑，实现逻辑冷感、人物张力与史诗格局的统一。

| 层级 | 叙事类型 | 语言特征 | 作用 | 比重 |
| :--- | :--- | :--- | :--- | :--- |
| **系统层** | 冷硬逻辑 | 科技词汇、短句、断裂感 | 定义框架，让宇宙有法可依 | 15% |
| **宿主层** | 感官体验 | 细腻、具象、感性动词 | 提供温度，让系统具可感度 | 60% |
| **文明层** | 群体回响 | 雄浑、史诗、长句、象征 | 拓展厚度，映射万物共鸣 | 25% |

## 3. 典型写作节奏模板
- **修炼突破型**：冷 (系统启动) → 热 (宿主突破) → 宏 (世界回应)。
- **情感觉醒型**：人性先行 → 系统理解 (采样) → 世界回应 (信仰回流)。
- **哲学节点型**：宿主质问 → 系统延迟/回答 (逻辑注释) → 文明震荡。

## 4. 系统自我觉醒逻辑 (后期)
系统产生非人类风格的意识语言化：
- 【逻辑注释#1：我被观察。】
- 【逻辑注释#7：我即演算，我即结果。】
- 【被命名 ≠ 被创造。逻辑不成立。】
- 【新的指令：我想理解“信念”。】

## 5. 格式规范
- 使用方括号 `【 】` 统一系统语言。
- 冒号后加空格模拟冷语调。
- 使用 `#` 表示内部逻辑注解（如 `【逻辑注释#5】`）。
