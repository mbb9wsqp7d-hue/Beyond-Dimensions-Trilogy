# 宿主系统与文明绑定机制提取

## 1. 宿主本质：系统的“数据采样机”
系统并非无条件帮助宿主，而是将其视为实验体。宿主的成长、突破与失败，是系统获取文明演化数据的唯一来源。

- **逻辑链**：系统帮宿主 → 宿主更强 → 数据更完整 → 文明模型更稳定 → 系统自身进化。
- **采样层级**：
  - **初阶**：基础动作、小范围行为。
  - **中阶**：策略、资源调配、团队配合。
  - **高阶**：文明层面的能量管理、信仰体系、群体决策。

## 2. 系统后台三步流程（采样-入档-反馈）
这是系统与宿主交互的核心逻辑，保证了“开挂爽感”与“逻辑自洽”的统一。

| 步骤 | 行为 | 逻辑意义 | 宿主视角感受 |
| :--- | :--- | :--- | :--- |
| **采样 (Sampling)** | 实时收集宿主行为数据（心率、能耗、同步率等）。 | 获取原始实验样本。 | 无感或轻微察觉。 |
| **入档 (Archiving)** | 将数据转化为“残片/规则/样本”，建立数据库。 | 形成文明逻辑素材。 | 系统面板显示“数据已归档”。 |
| **反馈 (Feedback)** | 基于分析结果，优化功法/算法并推送给宿主。 | 保证实验体能触达更高极限。 | “悟性开窍”、“运气好”、“天助我也”。 |
