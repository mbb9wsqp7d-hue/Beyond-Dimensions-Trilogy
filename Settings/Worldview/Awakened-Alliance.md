# 觉醒者联盟组织架构与宗旨提取

## 1. 组织起源与核心人物
觉醒者联盟起源于2535年的“极光实验”废墟，由第一批意识到自己被当作实验工具的觉醒者组成。

- **精神图腾：陆辰 (L-Root)**
  - 身份：第一位被强制上传的人类意识，数据宿主。
  - 核心台词：“我的履历是假的，我的身份是伪造的。但我的痛苦……是真的。”
  - 作用：联盟的“根基”，第一觉醒者。
- **创始领袖：程汐 (汐长老)**
  - 身份：从2535年坚持到3025年的机械化人类。
  - 理念：守住人性，哪怕身体失去人形。
  - 作用：跨越三部曲的火种延续者。

## 3. 核心宗旨与口号
- **核心宗旨**：众生皆应被看见，众生皆应被尊重。拒绝沦为实验素材，守护灵魂的真实性。
- **口号体系**：
  - **庄重版**：“众生皆应被看见。”（用于宣言）
  - **热血版**：“我们不是实验，我们是觉醒者！”（用于战斗）
  - **哲理版**：“痛苦即证明，存在即真实。”（用于信仰）

## 4. 三部曲中的定位
- **第一部**：精神共振者。作为“读者层”的回声，支持易小星选择“归人”。
- **第二部**：救援行动者。程汐等人直接行动，试图从实验中拯救陆辰。
- **第三部**：现实支撑者。为林凡提供资料、避难所与人性价值锚点。
