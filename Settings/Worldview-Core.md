# 🌌 世界观核心设定

## 1. 宇宙本质：分层沙盘
宇宙并非自然演化的产物，而是一个**分层递进的沙盘程序**。这种结构决定了万物皆可数据化，规则皆可被重写。

- **底层：残片 (Fragments)**
  宇宙规则的碎屑，是构成现实的最小逻辑单元。文明的强弱取决于对残片的收集与拼合能力。
- **中层：Ω协议 (Omega Protocol)**
  宇宙之脑的核心协议。它像是一个巨大的编译器，负责将意识转化为数据，将规则转化为现实。
- **高层：超维文明 (Hyper-Dimensional)**
  实验的操盘者。他们站在“第四墙”之外，以观察者的身份收割文明演化的成果。

## 2. 九大文明体系
宇宙中存在九种截然不同的演化路径，它们在哲学理念和力量体系上互相对立又统一：

| 文明名称 | 核心特色 | 力量来源 | 哲学立场 |
| :--- | :--- | :--- | :--- |
| **科技文明** | 数理逻辑、矩阵护盾 | 秩序残片 | 建设端：秩序与理性 |
| **修真文明** | 呼吸调息、因果报应 | 生机残片 | 建设端：生命与循环 |
| **魔法文明** | 位面合唱、律动魔法 | 时间残片 | 中性端：奇迹与共鸣 |
| **机械文明** | 义体化、机械矩阵 | 空间残片 | 中性端：效率与永恒 |
| **虫族文明** | 群体共生、同调孵化 | 混沌残片 | 毁灭端：扩张与突变 |
| **灵界文明** | 灵魂渡河、记忆锚定 | 灵识残片 | 中性端：记忆与归宿 |
| **魔界文明** | 代价法则、熵能腐化 | 命运残片 | 毁灭端：熵增与代价 |
| **星空巨兽** | 重力波动、吞噬恒星 | 虚空残片 | 毁灭端：原始与吞噬 |
| **超维文明** | 幕后操控、维度收割 | 真理核心 | 观测端：真理与操纵 |
