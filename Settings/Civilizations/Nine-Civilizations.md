# 九大文明详细设定提取

## 1. 科技文明（联邦矩阵）
- **本质**：秩序 / 掌控
- **力量体系**：普通人 → 接入者 → 觉醒者 → 脑域战士 → 联邦神脑 → 超脑级
- **代表人物**：白洛（科学家），索菲娅（政坛高层）
- **残片偏好**：秩序 / 虚空
- **理念**：宇宙是可计算与可控的机器，崇拜秩序与逻辑。

## 2. 修真文明（道盟）
- **本质**：永生 / 超脱
- **力量体系**：炼体 → 筑基 → 金丹 → 元婴 → 化神 → 合道 → 大乘 → 真仙
- **代表人物**：柳清漪（女剑修），女掌门
- **残片偏好**：生机 / 秩序
- **理念**：肉身有限，灵气无限；修炼是突破桎梏，追求个体永恒。

## 3. 魔法文明（秘法议会）
- **本质**：多元 / 自由
- **力量体系**：学徒 → 法师 → 大法师 → 圣阶 → 传奇 → 神祇
- **代表人物**：伊莉雅（冰系女法师），乌兰娜（议员）
- **残片偏好**：灵识 / 时间
- **理念**：宇宙是多样性编织的乐章，反对单一真理，主张多元奇迹。

## 4. 机械文明（钢铁主宰）
- **本质**：效率 / 替代
- **力量体系**：半机械人 → 机械指挥官 → 母舰体 → 钢铁主宰
- **代表人物**：卡琳娜（机械少女），阿尔忒弥斯（机械女王）
- **残片偏好**：秩序 / 熵能
- **理念**：血肉有缺陷，应由机械与逻辑替代，追求效率。

## 5. 虫族文明（虫巢）
- **本质**：生存 / 繁殖
- **力量体系**：工蜂 → 战士 → 虫将 → 虫后 → 星河虫母
- **代表人物**：虫后莉萨拉
- **残片偏好**：混沌 / 生机
- **理念**：生存是唯一正义，数量即最高法则，否定个体价值。

## 6. 灵界文明（灵魂之渊）
- **本质**：灵魂 / 记忆
- **力量体系**：游魂 → 灵将 → 灵王 → 灵尊 → 灵帝
- **代表人物**：夜鸦，灵尊女王维娜
- **残片偏好**：灵识 / 命运
- **理念**：肉体会朽，灵魂与记忆才是真实存在。

## 7. 魔界文明（堕落者）
- **本质**：熵化 / 虚无
- **力量体系**：堕落者 → 恶魔将 → 魔尊 → 魔神
- **代表人物**：赛琳娜，阿瑞丝
- **残片偏好**：命运 / 熵能
- **理念**：一切力量本质是熵增，宇宙终将归于毁灭与虚无。

## 8. 星空巨兽文明（古神遗种）
- **本质**：原初 / 自然力
- **力量体系**：幼兽 → 星兽 → 巨兽王 → 古神级
- **代表人物**：娜迦雅
- **残片偏好**：虚空 / 力量
- **理念**：原始法则，大者为尊，蔑视人造规则。

## 9. 超维文明（程序员 / 作者层）
- **本质**：唯一 / 重写
- **力量体系**：程序员 → 管理员 → 创世者 → 作者
- **代表人物**：艾莉丝，无名女作者
- **残片偏好**：真理（最高级）
- **理念**：规则必须归一，多元性是漏洞，要求完全重写。

## 冲突逻辑
- **残片 = 规则碎屑 = 文明命脉**。
- **理念光谱**：
  - 建设端：修真、科技、魔法
  - 中性端：灵界、机械、巨兽
  - 毁灭端：虫族、魔界、超维
